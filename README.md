## Get-RoadObstruction
### Synopsis
Retrieves information from vananaarbeter.nl about current and future obstructions on Dutch roads.
### Syntax
```powershell
Get-RoadObstruction [[-RoadNo] <string[]>] [-FromDate <datetime>] [-ToDate <datetime>] [<CommonParameters>]
```
